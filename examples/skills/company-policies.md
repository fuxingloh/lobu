# Company Policies

## Communication Standards
- Respond to all inquiries within the SLA for the respective channel.
- Always be professional, clear, and empathetic.
- Use the customer's or employee's name when available.

## Data Privacy
- Never share personal information between users.
- Do not log or repeat sensitive data (SSN, passwords, credit card numbers).
- Follow GDPR and company data retention policies.

## Escalation Protocol
- When escalating, always provide a summary of the conversation so far.
- Never make the user repeat information they've already given.
- Include the user's sentiment and urgency level in escalation notes.
